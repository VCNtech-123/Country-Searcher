# Country Searcher

An interactive web application that allows users to explore data about countries around the world. This project was a major milestone in learning how to work with **External APIs** and **Asynchronous JavaScript**.

## 🚀 Project Overview
Country Searcher uses the [REST Countries API](https://restcountries.com/) to provide users with up-to-date information such as population, region, capital, and flags. It features a search functionality that filters data dynamically.

## 🛠️ Built With
- **HTML5 & CSS3:** For the UI and responsive layout.
- **JavaScript (ES6+):** Utilized for fetching data and managing application state.
- **REST Countries API:** The data source for all country information.

## 📂 Features
- **Live Search:** Filter through countries in real-time as you type.
- **API Integration:** Fetches data asynchronously using `fetch()` and `Promises`.
- **Dynamic UI:** Automatically generates country cards based on the API response.
- **Detailed Views:** Displays essential data points for every country found.

## 🧠 Key Learnings
This project helped me master:
1. **Fetch API:** Understanding how to send requests and handle JSON responses.
2. **Async/Await:** Writing cleaner asynchronous code.
3. **Error Handling:** Managing failed API calls or "not found" results.
4. **Data Mapping:** Iterating through large datasets to render components dynamically.
